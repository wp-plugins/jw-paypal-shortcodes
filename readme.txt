Tool for easily inserting PayPal (Website Payments Standard) buttons.

== Description ==

Simple plugin for inserting PayPal (Website Payments Standard) ‘Add to Cart’ buttons with shortcodes. Includes visual editor button for easy inserting of shortcodes.

== Installation ==

Activate plugin. Set correct email address in Tools > PayPal Settings. Look for the new button in the visual editor to easily insert shortcodes wherever you like.
