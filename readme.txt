Tool for easily inserting PayPal (Website Payments Standard) buttons.

== Description ==

Simple plugin for inserting PayPal (Website Payments Standard) ‘Add to Cart’ buttons with shortcodes. Includes visual editor button for easy inserting of shortcodes.

== Installation ==

Activate plugin. Set correct email address in Tools > PayPal Settings. Look for the new button in the visual editor (looks like PayPal logo) to easily insert shortcodes wherever you like. The pop up will guide you through the button options.

You can also change the currency and language on the settings page.

Enter the absolute URL of images to be used in place of standard form buttons.
